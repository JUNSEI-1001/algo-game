# アルゴ - 数字推理カードゲーム アプリ設計ドキュメント

## アプリ概要

アルゴ（ALGO）は、相手の持つ数字カードを推理するカードゲーム。
白カード（0〜11）と黒カード（0〜11）の計24枚を使用し、相手のカードを当てることを目指す。

## キーユーザーフロー

### AI対戦フロー
1. ホーム → 「AI対戦」タップ
2. 人数選択（2人 / 4人）
3. カード配布アニメーション
4. ゲームボード表示
5. ターン制でアタック / パス
6. 全カード解明 → 結果画面

### ランダムマッチフロー
1. ホーム → 「ランダムマッチ」タップ
2. 人数選択（2人 / 4人）
3. マッチング待機画面
4. マッチング成立 → カード配布
5. ゲームボード（リアルタイム対戦）
6. 結果画面

### 合言葉ルームフロー
1. ホーム → 「合言葉ルーム」タップ
2. 「ルーム作成」または「ルーム参加」選択
3. 作成：合言葉表示 → 友達の参加待ち
4. 参加：合言葉入力 → ルーム入室
5. 全員揃ったらゲーム開始
6. ゲームボード → 結果画面

## アルゴ ゲームルール

### 基本ルール
- 白カード（0〜11）と黒カード（0〜11）の計24枚
- 各プレイヤーに4枚ずつ配布（4人の場合は各4枚）
- カードは昇順に並べ、同じ数字の場合は黒が右
- 自分のカードは見えるが、相手のカードは見えない

### ターンの進行
1. 山札から1枚引く
2. 引いたカードで相手のカードをアタック（数字を指定）
3. 正解：カードが表になり、もう一度アタック可能
4. 不正解：引いたカードを自分の手札に加え、ターン終了（またはパス）
5. 全カードが表になったプレイヤーは脱落

### 勝利条件
- 最後まで生き残ったプレイヤーが勝利
- または、相手の全カードを当てたプレイヤーが勝利

## コンポーネント設計

### AlgoCard
- `number`: 0〜11
- `color`: 'white' | 'black'
- `isRevealed`: boolean
- `isSelected`: boolean
- アニメーション：フリップ（表裏）

### GameBoard
- プレイヤーエリア（2人 or 4人）
- ターン表示
- アクションボタン

### AttackModal
- ターゲットカード選択
- 数字入力（0〜11）
- 確認ボタン

### MatchingScreen
- ローディングアニメーション
- プレイヤー数表示
